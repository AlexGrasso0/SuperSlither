Background

Overall goal is to make a user-friendly website game to help players waste time when bored. The game functions inculde game menu, game speed, high score system, controls (arrow keys), and webpage to host the game. Our motivation to create this game is that We personally like video games and are very passionate about video games so we figured why not create a user-friendly web game.
